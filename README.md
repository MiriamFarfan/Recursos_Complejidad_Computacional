
# 📐 Complejidad Computacional — Clase Interactiva

### Unidad 1: Fundamentos Matemáticos

> Aplicación web educativa interactiva basada en el material del curso de **Complejidad Computacional** impartido por el prof. Tonatiuh Hernández-Del-Toro.

---

## 🌐 Ver el sitio

🔗https://miriamfarfan.github.io/Recursos_Complejidad_Computacional/

---

## 📚 Contenido

La aplicación cubre los 8 módulos de la Unidad 1 en formato de video educativo interactivo:

- **Módulo 1 — ¿Qué es un Algoritmo?**
  - Definición formal, 5 propiedades, ejemplos del mundo real

- **Módulo 2 — Inducción Matemática**
  - Caso base, hipótesis inductiva, paso inductivo
  - Dominós animados, pruebas resueltas paso a paso

- **Módulo 3 — Definiciones Recursivas**
  - Factorial, Fibonacci, inducción fuerte, recursión estructural

- **Módulo 4 — Notación Asintótica**
  - Big-O, Big-Ω, Big-Θ, small-o, small-ω
  - Gráfica interactiva con slider de n

- **Módulo 5 — Análisis de Algoritmos**
  - Pseudocódigo animado línea por línea con costo al margen
  - Búsqueda binaria y Merge Sort visualizados

- **Módulo 6 — Grafos**
  - Matriz de adyacencia, lista de adyacencia, lista de aristas
  - Generación automática de representaciones

- **Módulo 7 — Ordenamiento y Recurrencias**
  - Teorema Maestro con sus 3 casos
  - Merge Sort O(n log n) con árbol de recursión

- **Módulo 8 — Quiz Final**
  - Preguntas integradoras de todos los temas
  - Sistema de puntuación acumulada ⭐

---

## ✨ Características

- 🎬 **Pseudocódigo animado** línea por línea con análisis de costo al margen
- 📊 **Gráfica interactiva** de complejidades con slider (O(1) hasta O(n!))
- 🧮 **Pruebas por inducción** paso a paso con bloques coloreados (Base / Hipótesis / Paso)
- 🎯 **Quiz por módulo** con retroalimentación inmediata y explicación
- 🕸️ **Grafo interactivo** con generación automática de matriz y lista de adyacencia
- 🔍 **Búsqueda binaria animada** con visualización del intervalo [lo, hi]
- 🔀 **Merge Sort animado** con árbol de recursión
- ⭐ **Sistema de puntuación** acumulada
- 🌙 **Tema pizarrón oscuro** con tipografía académica

## 🚀 Uso local

Solo descarga `index.html` y ábrelo en cualquier navegador moderno. No requiere instalación.

```bash
# Opción 1: abrir directo
open index.html

# Opción 2: servidor local
python3 -m http.server 8080
# → http://localhost:8080
```
